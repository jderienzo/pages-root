NIST Pages

This is the root of NIST's GitHub Pages-equivalent site.

This server has been created with a stripped-down version of 18F's pages-server code, which is intended to mimic the functionality of GitHub Pages (github.io) on a self-hosted server. This is required because the native GitHub functionality does not meet current Federal Government website requirements as well as the requirements of NIST's own IT Security Officer for hosting official web pages.

NIST users wishing to publish web sites using this service should first review the GitHub Rules of Behavior, and can find instructions for using it by searching the NIST internal web server for "github". NIST users wishing to add their site to the featured site list below should submit a pull request to the usnistgov/pages-root repo.

The source code for this page is available in the pages-root GitHub repository
